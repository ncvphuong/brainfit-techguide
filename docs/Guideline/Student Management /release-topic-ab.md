# Create and Assign Curriculum Templates AceyBrain

## 1. Purpose
This guideline provides step-by-step instructions to:
- Create Curriculum Templates
- Configure learning periods and topics
- Assign templates to students

---

## 2. Access System

1. Navigate to the system:
   https://acp.powerdigital.sg/acp/systems

2. Go to:
   - **AceyBrain**

   ![Image](./img/release-topic-ab-1.png)
   
   - **Curriculum Templates**

   ![Image](./img/release-topic-ab-2.png)
   

---

## 3. Create a New Curriculum Template

### Step 1: Create Template

1. Click **Add New Template**

   ![Image](./img/release-topic-ab-3.png)
   
2. Enter template name (e.g. `P1 - First Month`)

   ![Image](./img/release-topic-ab-4.png)
   
3. Click **Submit**

   ![Image](./img/release-topic-ab-5.png)
   

---

### Step 2: Configure Duration & Add Learning Periods

1. Input **Duration (days)**
   - Example: `31`

   ![Image](./img/release-topic-ab-6.png)
   
2. Confirm the duration. Click **+ Add Period**

   ![Image](./img/release-topic-ab-7.png)
   

3. Configure:
   - **Subject**
   - **Level**
   - **Topic**

   ![Image](./img/release-topic-ab-8.png)
   ![Image](./img/release-topic-ab-9.png)
   ![Image](./img/release-topic-ab-10.png)
   
> 💡 Tip:  
> You can select **"-- All Topics --"** to assign all topics at once instead of adding them individually.

4. Click **Add** to save the period

   ![Image](./img/release-topic-ab-11.png)
   
---

### Step 3: Adjust Number of Periods

1. Locate the period configuration

   ![Image](./img/release-topic-ab-12.png)
   
2. Edit the **number of periods**

   ![Image](./img/release-topic-ab-14.png)
   
3. Update the value as needed
4. Click **✔ (Check icon)** to confirm

   ![Image](./img/release-topic-ab-15.png)
   

---

## 4. Assign Curriculum to Student

### Step 1: Navigate to Student List

1. Select **Student**
2. Select **Student List**

---

### Step 2: Select Student

1. Find the student
2. Click **Show Game**

   ![Image](./img/release-topic-ab-16.png)
   

3. Click **Assign Curriculum**

   ![Image](./img/release-topic-ab-17.png)
   


---

### Step 3: Assign Template

1. Select:
   - **Start Date**

   ![Image](./img/release-topic-ab-18.png)
   

   - **Curriculum Template**

   ![Image](./img/release-topic-ab-19.png)
   


2. Click **Preview** to review assignment

   ![Image](./img/release-topic-ab-20.png)
   


3. Click **Confirm Assign**

   ![Image](./img/release-topic-ab-21.png)
   


---

### Step 4: Verify Assignment

1. Click **View Topics**

   ![Image](./img/release-topic-ab-22.png)
   

2. Validate:
   - Assigned topics
   - Start and end dates

   ![Image](./img/release-topic-ab-23.png)
   


> 📌 Note:
> - The system automatically distributes topics based on the selected template duration.
> - You can manually add more topics later to strengthen specific skills for the student.

---

## 5. Best Practices

- Use consistent naming convention for templates (e.g. `Grade - Duration`)
- Prefer **All Topics** when applying full curriculum coverage
- Always **Preview** before confirming assignment
- Review assigned topics to ensure alignment with student learning goals
