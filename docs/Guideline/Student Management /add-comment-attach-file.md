# Adding Comments for Student in BrainFit ACP
This feature is for SA, ML, CA, Trainer

## Steps to Add a Comment  

1. **Navigate** to [BrainFit ACP](https://acp.brainfitstudio.com/acp).  
2. Click **"Students"**.  

![Image](./img/add-comment-attach-file-1.png)

3. Type the **student's name**. 

![Image](./img/add-comment-attach-file-2.png) 

4. Click **"Search"**.  

![Image](./img/add-comment-attach-file-3.png)

5. Select the **student's profile**.  

![Image](./img/add-comment-attach-file-4.png)

6. Click **"Add Comment"**.  

![Image](./img/add-comment-attach-file-5.png)

7. **Type the comment** in the provided field. 

![Image](./img/add-comment-attach-file-6.png)

8. Click **"Choose File"** to attach a file. 

![Image](./img/add-comment-attach-file-7.png)

9. Click **"Submit"** to save the comment.  

![Image](./img/add-comment-attach-file-8.png)


## Steps to View and Edit a Comment  

10. Click **"Notes"**.  

![Image](./img/add-comment-attach-file-9.png)

11. Type the **student's name**.  

![Image](./img/add-comment-attach-file-10.png)

12. Click **"Search"**.  

![Image](./img/add-comment-attach-file-11.png)

13. Click to **view the note**.  

![Image](./img/add-comment-attach-file-12.png)

14. Click to **edit the note** if needed.  

![Image](./img/add-comment-attach-file-13.png)
