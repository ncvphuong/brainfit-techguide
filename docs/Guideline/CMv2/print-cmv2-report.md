# How to Access and Print Student CMv2 Report

## Steps to Access the Report  

1. Navigate to [ACP Portal](https://acp.brainfitstudio.com/acp).  
2. Click **"CogMaps V2"**.  

![Image](./img/print-cmv2-report-1.png)

3. Click **"Standard Score"**.  

![Image](./img/print-cmv2-report-2.png)


4. Click the **"Name..."** field.  

![Image](./img/print-cmv2-report-3.png)


5. Type the **student's name**.  
6. Click **"Search"**.  

![Image](./img/print-cmv2-report-4.png)



## Steps to Print the Report  

7. Click on the **student's name** you want to print.  

![Image](./img/print-cmv2-report-5.png)


8. Click the **"Round"** field.  

![Image](./img/print-cmv2-report-6.png)


9. Select the **round**.  
10. Click **"Show"**.  

![Image](./img/print-cmv2-report-7.png)


11. Click **"Print"**.  

![Image](./img/print-cmv2-report-8.png)


12. Select the **download folder** and **save** the file.
