This feature is for SA, ML, CA

## How to Input Student Assessment Data

1.  **Navigate** [ACP](https://acp.brainfitstudio.com/acp).
2.  Click **Class** in the navigation menu.

![Image](./img/input-assessment-1.png)

3.  Click the **Edit** button (or corresponding icon) for the specific class where you want to input assessment data.

![Image](./img/input-assessment-2.png)


4.  Click **Input Assessment**.

![Image](./img/input-assessment-3.png)


5.  In the **"Student Name..."** search field, type the name of the student you are looking for and then click **Search**.

![Image](./img/input-assessment-5.png)


6.  **Alternatively**, you can directly click the **Input** button located next to the student's name in the list below.

![Image](./img/input-assessment-7.png)


7.  Click on the specific **Area Assessment** you want to input data for.

![Image](./img/input-assessment-8.png)


8.  Click the dropdown menu within the assessment area to choose the desired option or value.

![Image](./img/input-assessment-9.png)


9.  Click the **"Comments"** field to type any relevant comments, if necessary.

![Image](./img/input-assessment-10.png)


10. Click **Save Assessment** to record the entered data.

![Image](./img/input-assessment-11.png)
