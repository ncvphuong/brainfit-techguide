# Delete Session Score

## Steps to Delete a Session Score  

1. **Login** with an admin account.  
2. Click **"Monthly Progress Reports"**.  

![Image](./img/delete-session-score-1.png)

3. Click **"Students"**.  

![Image](./img/delete-session-score-2.png)


4. Type the **name of the student** whose session score you want to delete. 

![Image](./img/delete-session-score-3.png)


5. Click **"Search"**.  

![Image](./img/delete-session-score-4.png)


6. **Select the student** from the search results.  

![Image](./img/delete-session-score-5.png)


7. **Select the time period**.  

![Image](./img/delete-session-score-6.png)


8. Click **"View"**.  

![Image](./img/delete-session-score-7.png)


9. Click **"Game Logs"**.  

![Image](./img/delete-session-score-8.png)


10. Click the **"Delete"** button to remove the session score.  

![Image](./img/delete-session-score-9.png)
